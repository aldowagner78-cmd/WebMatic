Inicio rápido

WebMatic sirve para grabar una tarea repetitiva en una página web y repetirla luego con un clic.

1. Abrí la página

Entrá al sitio donde querés automatizar una carga, búsqueda o trámite repetitivo.

2. Grabá la tarea

Presioná Grabar, hacé la tarea normalmente y luego presioná Detener.

3. Guardá la macro

Poné un nombre claro para reconocerla después.

4. Reproducí

Seleccioná la macro y presioná Reproducir.

Recomendación: empezá con macros cortas. Es más fácil corregir y combinar macros pequeñas que arreglar una macro muy larga.
🧭

Pantalla principal

La pantalla de WebMatic muestra tus macros guardadas y las acciones principales.

  • Grabar: comienza a registrar los pasos que hacés en la página.
  • Detener: finaliza la grabación o corta una reproducción en curso.
  • Reproducir: ejecuta la macro seleccionada.
  • ▶▶ Bucle: repite la macro varias veces.
  • Editar: abre la macro para revisar o corregir pasos.
  • Exportar / Importar: permite mover macros entre computadoras.
●

Grabar una macro

1

Entrá a la página donde querés automatizar una tarea.

2

Presioná Grabar.

3

Realizá la tarea normalmente: escribir, elegir opciones, hacer clic, buscar o avanzar.

4

Presioná Detener y guardá la macro con un nombre claro.

WebMatic intenta reconocer campos, botones, listas desplegables, esperas y navegación para que la macro funcione aunque la página tarde en cargar.
▶

Reproducir macros

  1. Seleccioná la macro en la lista.
  2. Presioná Reproducir.
  3. Esperá a que WebMatic termine.

Durante la reproducción aparece un indicador de progreso. Si algo falla, WebMatic muestra un mensaje con el paso, la acción y el selector relacionados.

No uses el mouse ni el teclado sobre la página mientras la macro se reproduce, salvo que necesites detenerla.
▶▶

Usar el botón ▶▶ Bucle

El bucle repite una macro varias veces. Es útil para cargas repetidas, consultas similares o tareas administrativas.

  1. Seleccioná la macro.
  2. Presioná ▶▶ Bucle.
  3. Indicá cuántas veces querés repetirla.
  4. Confirmá y dejá que WebMatic trabaje.
Si una macro navega a otra página, WebMatic puede continuar con la siguiente repetición cuando la página vuelve a estar lista.
Para evitar errores, probá primero una sola reproducción. Cuando funcione bien, usá el bucle.
✏️

Editar y corregir errores

El editor permite revisar la macro paso por paso.

  • Editor visual: muestra los pasos en forma más clara.
  • Editor IIM: muestra la macro en texto para ajustes precisos.
  • Editar macro: aparece cuando una reproducción falla y abre el paso relacionado.

Qué revisar cuando algo falla

  • Que la página sea la correcta.
  • Que el campo o botón exista en pantalla.
  • Que no falte una espera antes del paso.
  • Que el usuario haya iniciado sesión si el sitio lo requiere.
🔒

Datos sensibles

Cuando WebMatic detecta campos sensibles, como contraseñas o tokens, evita mostrar el valor real en la macro.

Es normal ver pasos redactados o protegidos. Eso ayuda a evitar que una contraseña quede escrita en una macro.
  • No guardes contraseñas reales en textos visibles.
  • No compartas macros que puedan contener datos personales.
  • Antes de exportar una macro, revisala si contiene información delicada.
⏱

Esperas y páginas dinámicas

Muchas páginas modernas tardan en mostrar campos, botones o resultados. WebMatic usa esperas para no ejecutar un paso antes de tiempo.

  • WAIT_FOR: espera hasta que aparezca un campo, botón o resultado.
  • WAIT: espera una cantidad fija de segundos.
  • Selectores robustos: ayudan a encontrar el campo correcto aunque cambie el identificador interno de la página.
Si una macro falla porque un campo no aparece a tiempo, agregá o ajustá un paso de espera antes de ese campo.
📦

Exportar, importar y mover macros

Podés guardar tus macros en un archivo para copiarlas a otra computadora o conservar una copia de respaldo.

Exportar

Guarda una macro o un conjunto de macros en un archivo.

Importar

Carga macros guardadas anteriormente.

Respaldo completo

Sirve para conservar todas tus macros antes de hacer cambios importantes.

Si compartís macros, revisá antes que no incluyan datos personales, números de expediente reales, usuarios o textos privados.
✅

Buenas prácticas

  • Usá nombres claros: por ejemplo, Buscar expediente o Cargar comprobante.
  • Probá la macro una vez antes de usar bucle.
  • No mezcles muchas tareas diferentes en una sola macro.
  • Si una página cambia mucho, revisá la macro antes de usarla en trabajo real.
  • Exportá tus macros importantes como respaldo.
🛟

Problemas frecuentes

La macro no encuentra un campo

Revisá si la página terminó de cargar. Si hace falta, agregá una espera antes del paso que falla.

La macro funciona una vez, pero falla en bucle

Probá primero una sola vuelta. Si cada vuelta cambia de página o estado, agregá una espera al inicio de la macro.

El sitio pide iniciar sesión

Iniciá sesión manualmente antes de reproducir la macro. Algunos sitios cierran la sesión después de cierto tiempo.

El password no aparece en la macro

Es correcto. WebMatic protege ese tipo de datos para que no queden visibles.

La página cambió y la macro falló

Editá la macro o grabala de nuevo. Algunos sitios cambian nombres internos, botones o estructura.
