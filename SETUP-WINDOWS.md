# Prompt para el agente de Windows

Copiá y pegá esto en el chat de GitHub Copilot en VS Code (Windows):

---

Cloná este repositorio en el Escritorio:

```
git clone https://github.com/aldowagner78-cmd/WebMatic.git
```

Luego cargá la extensión en Chrome o Edge:

- Ir a `chrome://extensions` (o `edge://extensions`)
- Activar "Modo desarrollador"
- Clic en "Cargar extensión sin empaquetar"
- Seleccionar la carpeta `WebMatic` (la que contiene `manifest.json`)

Eso es todo. No instales nada más.

Para sincronizar cambios en el futuro, abrí una terminal dentro de la carpeta `WebMatic` y ejecutá `git pull`, luego recargá la extensión en `chrome://extensions`.
